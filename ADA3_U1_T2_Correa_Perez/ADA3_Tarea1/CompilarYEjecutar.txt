*----------PASOS PARA COMPILAR Y EJECUTAR---------*
Para compilar el programa
1. Abrir la consola 
2. Trazar la ruta en donde se encuentra el archivo
3. Escribir javac Main.java, de la siguiente forma:
C:\ruta del archivo\javac Main.java
4. Para ejecular el programas solos se tiene que escribir java Main, de la siguiente forma:
C:\ruta del archivo\java Main

C:\Users\Audny Correa\Downloads\ADA3_Tarea1\ADA3_Tarea1\src>javac Main.java
